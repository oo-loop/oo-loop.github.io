Installation

Being a sass framework, Loop requires a sass environment to be setup

Setup options

  1. Via Sass applications or commande line
  2. Via node-sass npm install node-sass
  3. Via Webpack and sass loader npm install sass-loader node-sass webpack --save-dev
  4. Via Rollup and rollup-plugin-sass npm install rollup rollup-plugin-sass -D

Loop package

Install Loop via npm or yarn

npm install oo-loop / yarn add oo-loop

CSS property supports

Loop does not include any vendor prefixes by default.
Please use Autoprefixer to add more compatibilities to your project.
